# Task Scheduler

This project is organized into `Backend` and `Frontend` folders. The backend is built with Express.js.

## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm

### Setup

1. **Install dependencies**

   ```bash
   cd Backend
   npm install
   ```

2. **Configure environment variables**

   Edit the `.env` file in the project root. Example:
   ```
   PORT=3001
   ```

3. **Run the server**

   ```bash
   npm start
   ```
   or
   ```bash
   node src/index.js
   ```

4. **Access the API**

   Open [http://localhost:3001](http://localhost:3001) in your browser or use Postman.
